# 亚马逊联盟选品实战方法论：如何找到真正能赚钱的商品？

> 声明：本文仅代表个人观点，实践出真知。

---

## 一、为什么选品决定了你能否盈利？

很多新手做亚马逊联盟推广，一上来就急着投放广告，觉得"有曝光就会有成交"。结果呢？预算烧了一堆，转化却寥寥无几。

经过一系列的测试，我发现一个核心问题：**不是广告投放技术的问题，而是选品的问题。**

选品不当会导致四种典型的亏损场景：

**1. 商品本身没有市场需求**
有些商品本身搜索量、购买量就很低，你投放广告期待谁来买？这种商品即使排在第一位，也很难有转化。

**2. CPC太高，转化率无法覆盖成本**
有些商品搜索量、购买量都不错，但品牌词CPC极高。你用很高的出价去抢品牌词，最后发现佣金还不够覆盖广告成本。

**3. 联盟挂的不是最优商品**
这是最坑的一种情况。商品在联盟上挂出来了，但你深入一看，亚马逊上还有很多更优质的商品没在联盟挂。用户点击你的广告进去后，发现有更好的选择，于是购买了没挂佣金的商品。

**4. 为他人做嫁衣**
用户点了你的广告，进入详情页后，没买你推广的商品，而是买了这个商家其他没在联盟挂的商品。这种情况我亏损最多。

所以，选品的核心思路其实很简单：**就像我们自己 在亚马逊购物时如何选品，那么推广的时候就应该如何选品。**

---

## 二、两种购物场景，决定了你的投放策略

大家在亚马逊购物时，一般有两种情况：

**场景一：目标明确**
你知道自己要买什么品牌、什么型号，直接搜索，然后下单。比如你要买iPhone 15 Pro Max，你会直接搜索"iPhone 15 Pro Max"然后购买。

**场景二：目标模糊**
你并不清楚自己要买哪个品牌、哪个型号，需要在搜索结果中挑选。你会看评价、销量、评分，然后做出决策。

这两种购物场景，对应了我们联盟推广的两种投放策略：

| 购物场景 | 推广策略 | 适用情况 |
|---------|---------|---------|
| 目标明确 → 直接搜索品牌/型号 | 品牌词投放 | 用户有明确购买意向 |
| 目标模糊 → 挑选评测和销量好的 | 品类投放 | 用户还在比较选择 |

**核心原则**：用品牌词推广那些用户已经"想好了要买"的产品，用品类推广那些用户还在"犹豫不决"的产品。

## 三、选品的四个核心指标

既然选品如此重要，那我们应该按什么标准筛选？经过反复测试，我总结了四个核心指标：

### 1. 销量指标（上月销量）

**销量要求根据投放策略和价格分层：**

| 投放策略 | 价格区间 | 月销量最低要求 |
|---------|---------|--------------|
| 品牌词投放 | 单价较高（>$100） | ≥ 100 |
| 品类词投放 | 单价 < $300 | ≥ 1000 |

**逻辑**：品牌词投放的商品，用户已经有明确购买意向，月销量100说明商品有人认可；但品类词投放的商品，用户还在选择阶段，必须有更高的销量基础才能保证转化。

### 2. 评分指标

**最低门槛：评分 ≥ 4.0**

评分代表产品质量。低分商品不仅难卖，还会招来售后问题和差评，影响信任。

### 3. 评价数指标

**评价数根据投放策略和价格分层：**

| 投放策略 | 价格区间 | 评价数最低要求 |
|---------|---------|--------------|
| 品牌词投放 | 单价较高（>$100） | ≥ 100 |
| 品类词投放 / 低价 | 单价 < $300 | ≥ 500（新品除外） |

**逻辑**：高价商品或品牌词投放，评价数可以适当降低；但低价商品用品类词投放，必须有足够多的真实评价支撑用户决策。

### 4. 商家投放一致性（最容易被忽视）

**核心问题：这个商家在联盟上投放的商品范围，是否和亚马逊上主推的商品一致？**

如果一个商家在亚马逊上主推10款商品，但联盟上只挂了3款，说明他在"藏私"——好卖的他不想分佣金给你。这种情况不能选。

**正确做法：选择那些在联盟上投放商品比较多的商家，尤其是同一品类大量投放的商家。** 这样用户即使不买你推广的这一款，买了同商家的其他款，你也能拿到佣金。

## 四、价格分层：不同的价格区间，不同的投放策略

价格决定了用户的决策周期和投放策略。根据我的测试，我把商品分为三个价格区间：

### 1. 低价商品（<$100）

**特点**：客单价低、佣金低

**投放策略：品类关键词投放**

低价商品用品牌词投放是亏本的，因为品牌词CPC往往较高，佣金却只有几块钱，覆盖不了成本。

品类词投放的优点是搜索量大、CPC可能较低。即使GKP显示CPC是5-6元，但因为搜索量大，会有"溢出流量"让你的实际CPC压到1-2元甚至更低。

**但品类词有一个问题**：用户没有品牌忠诚度，他还在比较选择。所以低价商品的**性价比**非常重要——评价高、价格合适，才有可能转化。

**CPC控制：1-2元人民币**

**销量要求：月销量 ≥ 1000**（说明商品有竞争力）

### 2. 高价商品（>$300）

**特点**：用户决策周期长、考虑因素多

**投放策略：两个判断维度**

**维度一：商品本身的竞争力**
评价高、销量不错、竞争力OK，就可以投放。

**维度二：品牌词的搜索量**
这是关键指标。

- 如果品牌词搜索量 < 400/月，说明这个品牌基本没人搜，品牌词投放效果不会好，只能用品类词。
- 如果品牌词搜索量 > 1000-2000/月，说明这个品牌有热度，值得重点投放。

**注意事项：商品一致性**
高价商品尤其要检查联盟挂的商品是否和亚马逊主推的一致。如果用户在亚马逊上看到更便宜的没挂佣金的商品，你可能一笔佣金都拿不到。

**销量要求：月销量 ≥ 100**（高价商品销量门槛可以适当降低）

### 3. 最佳价格区间（>$100）

这是我认为最"香"的价格区间：

- 用户决策周期短
- 购买意向强
- 搜索量、销量通常较大
- 佣金能够覆盖广告成本

**投放策略：双投放**

这个区间既可以用品牌词，也可以用品类词。具体用哪种，看品牌词的搜索量和CPC情况：

- 品牌词搜索量大、CPC合理 → 品牌词投放
- 品牌词搜索量低或CPC太高 → 品类词投放

## 五、投放策略的选择逻辑

### 1. 什么时候用品牌词？

满足以下条件时，可以使用品牌词投放：

- **商品价格 > $100**
- **CPC可控**：在转化率5%的情况下不亏损
- **品牌词搜索量** ≥ 400/月（低于400效果不好）

**品牌词CPC评判标准**：最低CPC ≤ 7元人民币

品牌词的优点是用户意向明确，转化率高。缺点是CPC高、竞争大。

### 2. 什么时候用品类词？

满足以下条件时，使用品类词投放：

- **商品价格 < $100**
- **品牌词搜索量 < 400/月** 或 **CPC太高**
- **商品上月销量 > 500**（说明商品有竞争力）

**品类词CPC评判标准**：最低CPC ≤ 3元人民币

品类词的优点是搜索量大、CPC可能较低。缺点是用户没有品牌忠诚度，需要靠性价比转化。

### 3. 一个核心原则：看搜索量和性价比

无论用哪种策略，��要关注：

- **搜索量**：搜索量越大，CPC越有可能被"压低"
- **性价比**：用户凭什么是你的商品？评价、价格、销量是否足够有说服力？

---

## 六、性价比的评判标准

性价比是品类词投放的核心，那如何判断一个商品性价比高不高？

**核心指标：评价好 + 单价低**

具体看：

1. **评分**：至少 ≥ 4.3（优秀）
2. **价格**：在同类产品中有价格优势
3. **是否是 Top Seller**：如果是亚马逊自营或Top Seller标识，说明商品经过市场验证

**逻辑**：性价比高的商品，即使用户用品类词搜索，他的比较成本很低——因为你推荐的商品评价好、价格合理，不需要他花太多时间做决策。

## 七、两个隐藏的选品维度

### 1. 季节性和节日性

选品要**提前1-2个月布局**。

比如泳池机器人在4月份推广时ROI不错，但到了6月份就差了——因为4月份是准备季，6月份已经是旺季，CPC已经涨上去了。

**正确做法：提前预测接下来1-2个月可能热销的商品，提前布局。**

### 2. 品牌竞争力

检查两个方面：

- **商品一致性**：联盟挂的商品是否和亚马逊主推的一致？
- **品牌竞争力**：这个品牌在市场上的竞争力如何？有没有竞品分析？

如果能找到竞争力不错、搜索量OK、所有商品都在联盟投放的品牌，那就是**重点研究、重点投放、重点观察**的对象。

## 八、总结：选品检查清单

在决定投放一个商品之前，按这个清单检查：

| 检查项 | 品牌词投放 | 品类词投放 | 备注 |
|--------|----------|-----------|------|
| 价格 | > $100 | < $300 | 决定投放策略 |
| 月销量 | ≥ 100 | ≥ 1000 | 品类词要求更高 |
| 评分 | ≥ 4.0 | ≥ 4.0 | 基础门槛 |
| 评价数 | ≥ 100 | ≥ 500 | 低价商品要求更高 |
| 品牌词搜索量 | ≥ 400/月 | - | 决定能否用品牌词 |
| 品牌词CPC | ≤ ¥7 | - | 超过7亏本风险大 |
| 品类词CPC | - | ≤ ¥3 | 超过3挤压利润 |
| 商家投放一致性 | 联盟商品≈亚马逊主推 | 联盟商品≈亚马逊主推 | 避免为他人做嫁衣 |
| 性价比 | - | 评分≥4.3 + Top Seller | 品类词投放关键 |

### 投放策略快速选择

| 条件 | 推荐策略 |
|------|--------|
| 价格 > $100 + 品牌词搜索量 ≥ 400 + CPC ≤ 7 | 品牌词投放 |
| 价格 > $100 + 品牌词搜索量 < 400 或 CPC > 7 | 品类词投放（需销量 > 500） |
| 价格 < $100 | 品类词投放（需销量 > 1000 + 性价比高） |
| 价格 $100-$300 + 品牌词搜索量OK | 品牌词优先，否则品类词 |
