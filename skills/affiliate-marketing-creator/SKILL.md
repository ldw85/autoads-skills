# 联盟营销创意与广告素材制作

本skill提供完整的联盟营销创意挖掘和广告素材制作流程。

## 核心公式体系

### 1. PAS模型（痛点-焦虑-解决方案）

**Problem（问题）**：准确描述目标用户面临的具体问题
- 具体而非抽象
- 普遍而非个别
- 紧迫而非遥远

**Agitate（制造焦虑）**：放大问题带来的后果，层次递进：
1. 问题本身的痛苦
2. 问题带来的连锁反应
3. 不解决问题的长期后果
4. 错过解决机会的机会成本

**Solution（解决方案）**：在焦虑最高点提供解决方案
- 展示产品如何完美解决问题
- 提供具体效果和时间承诺
- 用证据支撑有效性
- 强调独特性

### 2. 双系统理论

**系统1（情感驱动）**：快速、直觉判断
- 激活方式：强有力的标题、抓眼球图片、情感化语言
- 目标：一两秒内让用户决定"继续阅读"或"立刻离开"

**系统2（理性支撑）**：缓慢、理性分析
- 激活方式：产品功能介绍、科学依据、用户评价
- 目标：打消疑虑，评估价值

**文案结构**：情感驱动 → 理性支撑

### 3. 需求层次理论

| 需求层次 | 产品类型 | 文案策略 |
|---------|---------|---------|
| 生理需求 | 健康、美容 | 强调功能性和效果 |
| 安全需求 | 保险、金融 | 强调可靠性和保障 |
| 社交需求 | 社交、时尚 | 强调归属感和认同感 |
| 尊重需求 | 奢侈品、教育 | 强调地位和成就 |
| 自我实现 | 技能培训、创业工具 | 强调成长和突破 |

**深层痛点挖掘方法**：连续追问"为什么"
- 例：减肥产品 → 表面痛点"超重" → 深层痛点"对人际关系和职业发展的担忧"

### 4. 高转化文案五要素

1. **强烈情感共鸣**：恐惧（失去）、愤怒（不满）、希望（憧憬）、骄傲（认同）
2. **清晰价值主张**：具体、可量化、可验证（如"30天减10斤"）
3. **可信证据支撑**：科学研究、专家推荐、用户评价、销售数据
4. **紧迫感和稀缺性**：利用损失厌恶心理
5. **明确CTA**：清晰告诉用户下一步做什么

## 工作流程

### 步骤1：产品分析
1. 确定推广产品
2. 识别目标用户画像
3. 挖掘用户深层痛点（追问"为什么"）
4. 确定对应的需求层次

### 步骤2：创意挖掘
1. 应用PAS模型构建文案框架
2. 结合双系统理论设计情感+理性内容
3. 根据需求层次选择文案策略
4. 生成多个创意方向

### 步骤3：素材制作（输出英文）
1. 撰写主标题（情感冲击）
2. 撰写副标题（价值承诺）
3. 制作正文（问题描述→焦虑放大→解决方案）
4. 添加证据支撑
5. 设计CTA

### 步骤4：落地页制作（可选）
使用 landing-page-generator 或 landing-page-converter skill

## 使用示例

用户要求推广一款健身App时：

1. **PAS应用**：
   - Problem："Want to get fit but can't stick with it"
   - Agitate："Watching others get healthier while you keep quitting"
   - Solution："This App helps you build a fitness habit in 21 days"

2. **双系统应用**：
   - 系统1：Before/after photos, motivational copy
   - 系统2：Scientific training plans, user success stories

3. **需求层次**：
   - 生理需求：Health
   - 社交需求：Approval through good physique

## 输出格式

所有输出使用英文，按照以下格式组织：
- Marketing Copy (Pas Model + Double System)
- Pain Points Analysis
- Headlines
- Body Copy
- CTA
