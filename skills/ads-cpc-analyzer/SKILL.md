# Google Ads CPC 分析技能

分析Google Ads广告账户，判断：
1. 哪些广告系列CPC应该**提升**
2. 哪些广告系列CPC应该**降低**
3. 哪些广告系列应该**扩大每日预算**

## 使用场景

当用户说：
- "分析广告账户"
- "CPC分析"
- "哪些广告应该提价/降价"
- "广告账户诊断"

## 工作流程

1. 获取账户所有ENABLED广告系列的花费和CPC
2. 获取对应ASIN的订单和销售额（从联盟API）
3. 获取ASIN价格（缓存 > 历史订单 > Amazon）
4. 计算最大合理CPC：`价格 × 7.5% / 50 × 6.8 × 0.9`
5. 判断：
   - **CPC应提升**：当前CPC < 最大合理CPC × 0.9
   - **CPC应降低**：当前CPC > 最大合理CPC × 1.2
   - **预算可扩量**：ROI > 100% 且订单量稳定
   - **应暂停/缩量**：连续2天无确认订单 且 花费 > $30

## 数据来源

- Google Ads API：广告花费、CPC出价
- Archer联盟API：Archer订单数据
- YeahPromos联盟API：YeahPromos订单数据
- 本地价格缓存：`asin_prices.json`

## 输出格式

```
=== 广告账户分析报告 ===

📊 CPC应提升的广告系列 (当前CPC偏低):
  - [广告名] (ASIN): 当前$XX, 最大合理$YY

📉 CPC应降低的广告系列 (当前CPC偏高):
  - [广告名] (ASIN): 当前$XX, 最大合理$YY

💰 应扩大每日预算的广告系列:
  - [广告名]: ROI XX%, 订单XX单

⚠️ 应暂停/缩量的广告系列:
  - [广告名]: 0订单, 花费$XX
```

## 注意事项

- CPC调整幅度建议每次不超过20%
- Pending订单按订单数 × 平均单价计入销售额
- 佣金率统一按7.5%计算
- 汇率按6.8计算
