# 广告文案公式库

## Headline 公式

### 数字 + 年份 + 品类
- "2026年最佳{品类}推荐"
- "TOP 10 {品类}年度精选"

### 用户群体 + 需求
- "程序员的效率装备清单"
- "送给男友的科技礼物"

### 问题导向
- "不知道选哪款？{品类}选购指南"
- "一篇搞懂{品类}：从入门到旗舰"

## CTA 按钮文案

### 高转化CTA
- "查看我们的首选" → 引导到最佳商品
- "查看所有推荐" → 锚点到商品列表
- "查看今日好价" → 强调价格优势
- "获取专属优惠" → 强调优惠

### 场景化CTA
- "通勤族首选"
- "游戏玩家标配"
- "音乐发烧友必入"

## Badge 标签文案

### 定位类
- 最佳综合 / 最佳性价比 / 最佳音质
- iOS用户首选 / 安卓用户必选
- 入门首选 / 旗舰之选

### 场景类
- 适合通勤 / 适合办公 / 适合运动
- 游戏神器 / 音乐利器 / 生产力工具

## Scarcity 紧迫感文案

- "仅剩X件库存"
- "优惠仅限本周"
- "价格随时调整"
- "Best Seller，热销中"
